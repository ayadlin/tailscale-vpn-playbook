# Contributing

## Goals
- Keep this repo **copy/paste-first**.
- Prefer **explicit variables** over hidden defaults.
- Document tradeoffs (routing + DNS) and failure modes.

## PR checklist
- [ ] Docs updated if behavior changes
- [ ] Scripts are idempotent where possible
- [ ] `shellcheck` passes for Bash scripts
- [ ] PowerShell scripts pass PSScriptAnalyzer
- [ ] Mermaid diagrams updated if architecture changes
